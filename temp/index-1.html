Ghostty

ghostty.org

Native terminal.

Terminal emulators
Our review

What we like

Core written in Zig with native interface for macOS (SwiftUI) and Linux (GTK). Extremely fast performance using low-level graphics e.g. Metal on macOS. Zero config by default. Supports iTerm themes. Native tabs, splits, and OS-level features e.g. right click system context menu on Mac. Quake mode global shortcut.

What we don't like

Might work on Windows, but no official support yet.

Reviewed: 2025-01-02
