Some takeaways:

  1. When delivering tough feedback, use this framework:

    1. Start directly: “I want to chat with you about [specific issue]”

    2. Share observations: “What I’m hearing/seeing is [specific examples]”

    3. Connect to impact: “We both know that to be successful here...”

    4. Focus on future: “Let’s discuss how we can improve this going forward”

  2. If someone gets defensive during feedback:

    1. Pause the conversation: “Let’s take a moment here”

    2. Reinforce positive intent: “I’m sharing this to help you succeed”

    3. Acknowledge emotions: “I can see this is triggering strong feelings”

    4. Offer space: “We can continue now or pause and resume later”

  3. A leader’s job isn’t to make everyone happy—it’s to create conditions for success. This means:

    1. Setting clear expectations

    2. Having difficult conversations when needed

    3. Building a culture of accountability

    4. Focusing on results over popularity

    5. Creating an environment where high performers can thrive

  4. End every meeting by answering three key questions:

    1. What did we decide here? (Go around the room to ensure alignment)

    2. Who needs to do what by when? (Clear action items with owners and deadlines)

    3. Who else needs to know? (Communication plan for decisions)

  5. Before starting a company together, co-founders should create a “founder prenup.” This entails having hard conversations about:

    1. Personal values and how they align

    2. Vision for what success looks like

    3. How to handle conflict and disagreements

    4. Desired company culture
