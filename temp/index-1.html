Shu Ding

I am a designer and developer based in Berlin. My interests span a broad spectrum of subjects, encompassing web development, creative coding, game design, and human-computer interaction.

Since high school, I’ve spent years on algorithm competitions, freelance web design and development. In 2017, I earned my BSc degree in Computer Science from Fudan University in Shanghai, and joined as a software engineer at Microsoft. In 2018, I joined Vercel to help build the next generation of web development tools.

You can gain further insights into my background and interests through my thoughts, projects, find me on GitHub, Twitter, Instagram, or LinkedIn.

Tips & Tools

The latest community-built utilities and integrations to help you build with Astro:

  • Domain Expansion - an Astro integration that adds support for incremental builds.
  • Sonda - visualize your Astro bundles
  • astro-word-loader - load .docx and .doc files into an Astro Content Collection
  • astro-typed-links - an Astro integration that automatically get typed links to your pages
  • astro-globals - an Astro integration that stores site data in a global virtual module
  • @terrahq/astro-core - a collection of core Astro components designed to streamline asset management and simplify form creation
  • Flickr loader - a content layer loader for the Flickr API
  • github-repos-astro-loader - a content loader for Astro that fetches GitHub repositories and their README files
  • starlight-plugin-show-latest-version - show the latest released version of your Starlight plugin in your plugin’s Starlight documentation
  • Bag of Tricks: <TurnSignal /> - set individual, per-link view transitions and choose different animations for each navigation

… and some built with Astro:

  • RSS Beauty - Transform RSS feeds into beautiful reading formats for a better reading experience.
  • Astro glob() Loader Config Generator - Need to check your content collections loader configuration? Use this web app to generate the correct settings when using Astro’s built-in glob() loader.
  • OG image maker - A free OG image generator tool you can use to create beautiful Open Graph images for your blog posts, articles, and social media posts.
  • Two idiots, one keyboard - an engineering team-bonding game inspired by ThePrimagen “two idiots one keyboard” video
