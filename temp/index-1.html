    AspireStarterPack for go

    Description

    This is a starter pack for go projects.

    Usage

    To use this starter pack, run the following command:

    dotnet restore
cd AppHost
dotnet run

    Default API

    echo

    • GET: http://localhost:5000/echo?name=world

    counter

    • GET: http://localhost:5000/count
