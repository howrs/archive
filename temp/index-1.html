    Nukesor/README.md

    Hello there, I'm Arne

    I make things! Here're a few projects:

    • 🌠 pueue: A shell command scheduler.
    • 🔶 comfy-table a rather popular CLI table library.
    • ⚙️ Bois an opinionated system config and dotfile manager (still WIP).
    • ❤️ infrastructure hosting for the Tiefgang collective.

    I made things! Here're the most prominent discontinued projects (due to various reasons):

    • 🤖 Ultimate poll bot: A chat bot for all kinds of voting, polls and doodles with 2 million all-time users in the end.
    • 🤖 Sticker Finder: The solution to tag, manage and find your favorite stickers.

    I also like good tooling! These aren't built with public usage in mind, though:

    • pueue-webhook-server: A webhook server I use to deploy my own stuff.
    • geil: A helper to manage too many git repositories.
    • game-saver: Save scumming for games I'm too impatient for.
    • hetznerbot: Sniping the best deals on Hetzner's server auction house.
    • scripts: My personal scripts.
