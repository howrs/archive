      ======= imjuni/README.md

      TypeScript lovers and OpenSource developer. I develop TypeScript tools for boost productivity. For example, automatic create JSONSchema from TypeScript interface or automatic create index.ts file etc.

      Stats

      Tool & Library

      Name Status Desc.
      create-ts-index CLI to generate index.ts file for webpack, rollup entrypoint
      ctix CLI to generate index.ts file for webpack, rollup entrypoint
      simple-tjscli CLI to generate JSON schema from TypeScript interface
      schema-nozzle CLI to generate JSONSchema from TypeScript interface, type alias, class and enum
      fast-maker CLI to generate route configuration using directory structure
      erdia CLI to generate mermaid.js ER diagram using TypeORM entity
      jin-frame Library to define RESTful API call mechanics using decorator, class
      deps-diff CLI to generate commit log that dependency difference between previous commit and current commit
      my-easy-fp Library to misc programming tool, like populate array and chunk, sleep etc.

      my-only-either Library to misc programming tool, like exist, getDirname, isDirectory, isEmptyDir etc.
