I wrote my own because I failed to find one I liked:

https://github.com/walles/px
Lots of unique (?) features:

  • For “python3 hello.py“ ptop shows “hello.py“ (helpful) rather than “python3“ (unhelpful). Works for lots of VMs, ruby, node, java, …
  • ptop shows you the top CPU using processes in a stable list (at least htop doesn’t do this)
  • ptop shows short lived processes, useful if one program (like make or cargo) spawns lots of subprocesses
  • ptop’s UI doesn’t break down when your system has more than just a few cores
  • …
See web page for details, or download the 55kb binary and try it for yourself :).
